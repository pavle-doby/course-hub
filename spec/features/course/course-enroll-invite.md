# Feature: Course Enroll Invite

**Status:** Draft

## Overview

Allows a course creator (or authorized user) to invite others to enroll in a course via a shareable invite link. If the recipient already has a Course Hub account, they are enrolled immediately and receive an in-app notification. If they do not have an account, they are directed to sign up first and are then automatically enrolled in the course upon completing registration.

## User Stories

- As a **creator**, I can generate an enroll-invite link for my course so that I can share it with anyone I want to invite.
- As a **creator**, I can copy the enroll-invite link to my clipboard so that I can easily send it via any channel (email, chat, etc.).
- As an **existing user**, when I open an enroll-invite link I am automatically enrolled in the course so that I can start following it without any extra steps.
- As an **existing user**, I receive an in-app notification when I am enrolled via an invite link so that I am aware of the new course in my list.
- As a **visitor** (not yet registered), when I open an enroll-invite link I am directed to the sign-up flow so that I can create an account and be enrolled in the course automatically upon completion.
- As a **new user**, after completing sign-up and onboarding I am redirected directly to the course I was enrolled in via the invite link so that I can start the course without having to find it manually.
