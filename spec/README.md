# Course Hub - Specifications

This document outlines the specifications for the Course Hub application, which serves as platform that enables creators to share their courses and allows users to discover and enroll in courses of their interest.

## Feature Specs

| Spec                                                     | Status   | Description                                                                                        |
| -------------------------------------------------------- | -------- | -------------------------------------------------------------------------------------------------- |
| [Sign Up](features/auth/auth-sign-up.md)                 | Approved | Register a new account with email and password                                                     |
| [Log In](features/auth/auth-log-in.md)                   | Approved | Authenticate an existing account with email and password                                           |
| [Onboarding](features/auth/auth-onboarding.md)           | Draft    | Post sign-up profile setup flow with redirect to home or enrolled course                           |
| [Create Course](features/course/course-create.md)        | Draft    | Build a new course with Topics and Lessons using a tree-structured editor                          |
| [Enroll in a Course](features/course/course-enroll.md)   | Draft    | Enroll in a course and track per-block progress with daily, weekly, monthly, and overall summaries |
| [Enroll Invite](features/course/course-enroll-invite.md) | Draft    | Invite users to enroll in a course via a shareable link, with auto-enroll on sign-up for new users |
